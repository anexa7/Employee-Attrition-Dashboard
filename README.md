# Employee Attrition Analysis Dashboard
## Live Dashboard
[View Dashboard Here](https://anexa7.github.io/My_Project/)

This project analyzes employee attrition patterns using Power BI.  
It focuses on identifying the main factors influencing employee turnover, such as job satisfaction, work-life balance, overtime, salary band, and performance rating.

## Project Files
- **Power BI Dashboard** – main dashboard file
- **DATASET.xlsx** – dataset used for analysis
- **Project Report.pdf** – detailed report
- **Project Synopsis** – project summary
- **AICW PPT.pptx** – presentation file
- **Video Demonstration** – project demo video

## Live Dashboard
[Click here to view the dashboard online](https://anexa7.github.io/My_Project/)

## Key Insights
- Attrition varies by job satisfaction score
- Overtime has a strong impact on attrition
- Work-life balance influences employee retention
- Salary band and performance rating also affect attrition trends
